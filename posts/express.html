getting started :

Let's get started with a short explanation of the whole workflow of the frontend and the backend of a website or web application.

To create a full stack application we need to have two things, the frontend of the application, that we build using HTML, CSS, JS and React. and the second thing is the backend of the application, that's built using the following technologies: Node JS, Express and MongoDB for storing data.

We use a third party package called mongoose (object data mapper) to interact with MongoDB database. moongose allows us to communicate easily with the database and perform CRUD operations using the built-in methods of mongoose.

So, to get data from the database we need to make an HTTP Request (from the browser to the server) to an Endpoint ( route ) that is created with Express . There is a different methods of HTTP Request, but the most important four are : GET, POST, PUT and DELETE. Once we make that request the server will communicate with the database and get the JSON data and send it back to the browser. and then the browser dispaly this data to the user.
